---

##  Features

###  Authentication
- Google OAuth Login
- Session-based user persistence
- Secure access to dashboards

###  Dashboards
- Separate dashboards for:
  - JEE
  - NEET
  - UPSC
  - GATE
- Study streak display
- Course progress visualization
- Weekly study graphs (Chart.js)

###  Online Quiz Engine
- Subject & exam-wise quizzes
- Randomized questions
- Instant answer feedback
- Correct answer highlighting

###  Flashcards
- Flip-card UI with 3D animation
- Keyboard navigation
- Exam-wise flashcard decks
- Randomized order

###  YouTube Learning Hub
- Subject-wise curated YouTube search links
- NEET / JEE / UPSC segregation
- Beautiful glass-morphism UI

###  PYQs (Previous Year Questions)
- Year-wise PDFs
- JEE / NEET / UPSC support
- External trusted sources (Mathongo / Archive)
